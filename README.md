# parallel.kevin.energy

Three pieces on a new category in AI infrastructure — an essay, a workflow demo, and a comparative battlecard. Built in May 2026 while preparing for a Technical Product Marketing Manager interview at [Parallel Web Systems](https://parallel.ai).

Live: **[parallel.kevin.energy](https://parallel.kevin.energy)**

## Structure

```
index.html              Landing page
essay/index.html        01 — The Agent Loop Needs Plumbing, Not Snippets
demo/                   02 — Workflow demo (in progress)
battlecard/             03 — Comparative battlecard (in progress)
tests/run-tests.js      Voice + structure + link-integrity tests
```

## Why

The essay argues that "web infrastructure for agents" is becoming a distinct product category, separable from web search. The demo shows what an agent built on that primitive looks like end-to-end. The battlecard audits the loudest APIs on the claim.

Together they're the first-week deliverables for a TPMM working on the category — shipped before the interview, so the conversation can start a layer up.

## Run locally

```bash
open index.html
node tests/run-tests.js
```
